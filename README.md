# 📝 File Creator (Node.js CLI Tool)

A simple command-line based File Creator built using Node.js.  
This tool allows users to create files dynamically by entering a filename and content via terminal input.

---

## 🚀 Features

- Create files from terminal input
- Accept custom filename
- Accept file content from user
- Automatically generates `.txt` file
- Displays success message after file creation

---

## 🛠️ Tech Stack

- Node.js
- JavaScript (ES Modules / File System module)
